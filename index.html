Discord Bot


Click to see the repository!

This discord bot was made not only to make my school discord server more lively, but also to learn more about the discord API. Rather than importing an existing bot, I felt that creating my own bot from scratch was a good learning opportunity. It could hence be customised to account for the needs of the server. Below shows an example of the bot returning a 'hi', based on your username, and displays some of the core commands this bot has in store.

This bot was developed during the women's UEFA Euros. Therefore, I introduced an easy way to retrieve football scores by the comfort of the discord channel. All football matches, their scores, and their updates for any day could be retrieved by typing &weuro <date in July> as shown below

In addition to this, the bot calls on other APIs such as a weather API that retrives current weather conditions and weather forecasts for any given location. Other features include:-
Asynchronous timers, asynchronous reminders that pings the user at the specified timer with the set message, conversion between celcius and fahrenheit and vice versa, currency conversion, simple web searches, ability to add commands within a channel, 8ball machine, and others!
